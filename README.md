# AE2 BlueMap NBT Fix
This is a work in progress addon for BlueMap which will fix some invisible AE2 blocks.

Important note for future self: BlueMap doesn't index textures unless a model explicitly references them, so be sure to reference the textures you want to use in at least one model's json file. This probably isn't relevant for most mods, but because AE2's cable rendering is so unconventional, it doesn't explicitly define any textures to use in it's model files.
